# Discord Map Bot

Bu bot, veritabanındaki şehirleri harita üzerinde işaretler.

## Komutlar

!city ŞehirAdı  
→ Belirtilen şehri haritada gösterir.

!allcities  
→ Veritabanındaki tüm şehirleri haritada gösterir.

## Gereksinimler

- Python 3.10+
- discord.py
- matplotlib

## Kurulum

pip install -r requirements.txt  
python bot.py
